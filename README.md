#LinkEl JQuery Plugin#

**LinkEl** is a simple jQuery plugin to attach links to elements

## Version ##
0.2.0

## Usage ##
    $("span").linkel({
		href: 'http://www.test.com',
		text: 'Click Here',
		target: '_blank'
	});

## Vendors ##
jQuery - [http://jquery.com](http://jquery.com)
